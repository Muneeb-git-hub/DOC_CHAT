# DOC_CHAT
Medical Q&A chatbot built with LangChain + FAISS + LLaMA 3.1 — answers only from your uploaded documents, no hallucinations. Upload your medical PDFs, ask any question, and get accurate answers strictly from your own data. Built using Retrieval-Augmented Generation (RAG) to ensure responses are always grounded in context. Powered by Meta LLaMA 3.1 8B Instruct via HuggingFace and semantic search using sentence-transformers. Features a clean Streamlit chat interface with session memory for smooth conversations. Embeddings are stored locally using FAISS for fast and efficient retrieval. Designed for students, researchers, and healthcare enthusiasts who want AI answers from trusted medical sources.
